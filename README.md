# Football-API CLI

This is a CLI application to fetch football api data from football-api hosted in Rapid-Api.

### RapidAPI - Football API Url

### Build

`script make football-api `

### Available Commands

- league
- standings
- team
- fixtures
- fixture-event
- fixture-lineup
- player-stat
- top-scorer
- help
